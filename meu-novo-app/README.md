# 🚀 RefatoreMe - EAD do Futuro

Este é um projeto de uma plataforma de ensino a distância (EAD) moderna, desenvolvido com **React** e **Vite**. O objetivo é oferecer uma interface limpa e responsiva para cursos online.

## 🛠️ Tecnologias Utilizadas

- **React 19**: Biblioteca para construção da interface.
- **Vite**: Ferramenta de build rápida para o desenvolvimento.
- **Lucide React**: Biblioteca de ícones elegantes.
- **CSS3**: Estilização personalizada e responsiva.
